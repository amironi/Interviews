
1. Open a Visual Studio command prompt.

2. Compile the rtc.cpp file without any command-line options:

	cl rtc.cpp

3. Run the resulting rtc.exe program. Input "Hello World" and watch the application go down in flames.

4. Run the program again. Input "Hello" and watch the application issue no output.

5. Compile the rtc.cpp file with debugging information and runtime checks enabled:

	cl /Zi /RTC1 rtc.cpp

6. Run the program with "Hello World". Attach a debugger when it crashes and inspect the call stack.

7. Run the program with "Hello". Attach a debugger when it crashes and inspect the call stack.
